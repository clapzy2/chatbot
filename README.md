# 📚 TextBot — локальный ИИ-ассистент по текстам

Бот отвечает на вопросы по твоим документам. Работает полностью офлайн — никаких платных API.

---

## 🖥️ Что нужно скачать

### 1. Python 3.10 или новее
Скачать: https://www.python.org/downloads/

> ⚠️ При установке обязательно поставь галочку **"Add Python to PATH"**

### 2. Ollama (запускает языковую модель)
Скачать: https://ollama.com/download

Установи как обычную программу. После установки **перезапусти компьютер**.

### 3. Git (для клонирования репозитория)
Скачать: https://git-scm.com/downloads

---

## 🚀 Установка — шаг за шагом

### Шаг 1 — Скачать модель
Открой командную строку (cmd) и введи:
```
ollama pull qwen3:8b
```
Скачается ~5 ГБ. Подожди до завершения.

> Если мало RAM (меньше 8 ГБ), используй лёгкую версию:
> ```
> ollama pull qwen3:4b
> ```

---

### Шаг 2 — Клонировать проект
```
git clone https://github.com/clapzy2/chatbot
cd chatbot
```

---

### Шаг 3 — Создать виртуальное окружение
```
python -m venv .venv
```

Активировать (Windows):
```
.venv\Scripts\activate
```

После активации в начале строки появится `(.venv)` — это значит всё правильно.

---

### Шаг 4 — Установить зависимости
```
pip install -r requirements.txt
pip install -U langchain-huggingface
```

Займёт 3–5 минут.

---

### Шаг 5 — Добавить свои файлы
Положи документы (PDF, TXT, DOCX, EPUB...) в папку `docs/` внутри проекта.

---

### Шаг 6 — Проиндексировать документы
```
python ingest.py
```

При **первом запуске** скачается embedding-модель (~560 МБ) — это нормально, подожди.
В конце должно появиться: `✅ Готово!`

---

### Шаг 7 — Запустить бота
```
python main.py
```

Автоматически откроется браузер на `http://127.0.0.1:7860`

Если браузер не открылся — зайди вручную по этому адресу.

---

## 💬 Как пользоваться

1. Открой вкладку **«Файлы»** → нажми **«Индексировать папку docs/»**
2. Перейди на вкладку **«Чат»**
3. Задавай вопросы по своим текстам

В dropdown над чатом можно выбрать конкретный файл — бот будет искать только в нём.

## ❓ Частые проблемы

**Ошибка 502 при запуске**
Добавь в начало файла `main.py` после строки `import os`:
```python
os.environ["NO_PROXY"] = "localhost,127.0.0.1"
os.environ["no_proxy"] = "localhost,127.0.0.1"
```

**`python` не найден**
Попробуй `python3` вместо `python`.

**Ollama не запускается**
Убедись что Ollama установлена и открыта (должна быть иконка в трее).
Или запусти вручную в cmd: `ollama serve`

**Медленные ответы**
Это нормально для локальной модели. Qwen3-8B отвечает ~15–30 секунд.
Для ускорения в `config.py` поставь `USE_HYDE = False`.

## ⚙️ Смена модели

В файле `config.py` найди строку:
```python
OLLAMA_MODEL = "qwen3:8b"
```
И замени на любую другую модель из Ollama.

| Модель | RAM | Качество |
|--------|-----|----------|
| qwen3:4b | 5 ГБ | Хорошее |
| qwen3:8b | 8 ГБ | Отличное ⭐ |
| qwen3:14b | 14 ГБ | Максимум |
