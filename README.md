# Бонч База Знаний

Многопользовательская система для работы с учебными текстами на основе RAG (Retrieval-Augmented Generation). Задавай вопросы по загруженным документам — бот находит ответ в тексте и подтверждает цитатами.

## Возможности

- **Точные ответы** по загруженным текстам (PDF, TXT, EPUB, DOCX, FB2, HTML)
- **Контекстный чанкинг** — автоматически определяет разделы/главы в тексте
- **Фильтр по разделам** — "Ответь из Речи Федра" → ищет только в нужном разделе
- **HyDE** — переформулирует вопрос для лучшего поиска
- **Реранкер** — уточнение результатов для максимальной точности
- **Режим экзамена** — генерация вопросов и проверка ответов студентов
- **Стриминг** — ответ печатается в реальном времени
- **Мультипользовательский** — у каждого своя история чата
- **Автономный режим** — работает без интернета через Ollama

## Стек

| Компонент | Технология |
|-----------|-----------|
| LLM | Qwen3-32B через OpenRouter API / Ollama |
| Эмбеддинги | BAAI/bge-m3 (мультиязычный, 1024 изм.) |
| Реранкер | BAAI/bge-reranker-v2-m3 |
| Векторная БД | ChromaDB |
| GUI | Gradio |

## Быстрый старт

### 1. Клонируй репозиторий
```bash
git clone https://github.com/clapzy2/chatbot.git
cd chatbot
```

### 2. Установи зависимости
```bash
python -m venv .venv
.venv\Scripts\activate      # Windows
# source .venv/bin/activate  # Linux/Mac
pip install -r requirements.txt
```

### 3. Получи API ключ
- Зарегистрируйся на [OpenRouter](https://openrouter.ai)
- Создай API ключ в [Settings → Keys](https://openrouter.ai/settings/keys)
- Вставь ключ в `config.py`:
```python
API_KEY = "sk-or-..."
```

### 4. Запусти
```bash
python main.py
```

Бот откроется в браузере на `http://localhost:7860`.

### 5. Загрузи тексты
- Загрузи через вкладку "📖 Файлы" в интерфейсе
- Или положи файлы в папку `docs/` и нажми "Индексировать"

## Конфигурация

Всё настраивается в `config.py`:

| Параметр | Описание | По умолчанию |
|----------|----------|-------------|
| `LLM_MODE` | `"api"` (облако) или `"ollama"` (локально) | `"api"` |
| `API_MODEL` | Модель на OpenRouter | `"qwen/qwen3-32b"` |
| `CHUNK_SIZE` | Размер фрагмента в символах | `1200` |
| `RETRIEVAL_TOP_K` | Сколько кандидатов искать | `20` |
| `RERANK_TOP_K` | Сколько оставить после реранкинга | `7` |
| `USE_HYDE` | Включить переформулировку | `True` |
| `GUI_SHARE` | Публичная ссылка | `True` |

## Автономный режим (без интернета)

Установи Ollama и скачай модель:

```bash
# Установи Ollama: https://ollama.com
ollama pull qwen3:8b
```

В `config.py`:
```python
LLM_MODE = "ollama"
OLLAMA_MODEL = "qwen3:8b"
```
