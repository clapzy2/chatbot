# Бонч База Знаний

Многопользовательская система для работы с учебными текстами на основе RAG.

## Возможности

- Точные ответы по загруженным текстам (PDF, TXT, EPUB, DOCX, FB2, HTML)
- Контекстный чанкинг — автоматически определяет разделы/главы
- Фильтр по разделам — поиск ограничен выбранным разделом
- HyDE — переформулирует вопрос для лучшего поиска
- Реранкер — уточнение результатов
- Стриминг — ответ печатается в реальном времени
- Автономный режим — работает без интернета через Ollama

## Быстрый старт

1. git clone https://github.com/clapzy2/chatbot.git
2. python -m venv .venv && .venv\Scripts\activate
3. pip install -r requirements.txt
4. Создай файл .env: API_KEY=sk-or-ваш-ключ
5. python main.py

## Автономный режим

ollama pull qwen3:8b
В config.py: LLM_MODE = "ollama"
